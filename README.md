# timp-746_kas
746_kas-2-1
На вход подаётся число n ≤ 2147483647, а также n чисел x, |x| ≤ 2147483647 для
любого целого i от 1 до n.
Вывести значение среднего арифметического от всех x.
Вещественный тип данных.
